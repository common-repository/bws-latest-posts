Add latest posts to your WordPress website widgets. Display your posts anywhere in your content.

== Description ==

Simple plugin which adds latest posts widget to your WordPress website. Show date, author, reading time, featured image, excerpt, and comments number. Display latest posts from particular category only.

Highlight latest posts for your visitors!

https://www.youtube.com/watch?v=vLYn3XIo1dk

= Features =

* Add latest posts to widgets
* Change widget title
* Display latest posts from a certain category
* Customize Read More text
* Choose the number of latest posts to display
* Set the excerpt length
* Set default image if post image is missing
* Display additional post info:
	* Date
	* Author
	* Reading time
	* Comments number
	* Featured image
	* Excerpt
* Add custom code via plugin settings page
* Compatible with latest WordPress version
* Incredibly simple settings for fast setup without modifying code
* Detailed step-by-step documentation and videos

== Installation ==

1. Upload the `bws-latest-posts` folder to `/wp-content/plugins/` directory.
2. Activate the plugin using the 'Plugins' menu in your WordPress admin panel.
3. Go to Appearance -> Widgets administration page and look for the widget named Latest Posts Widget
